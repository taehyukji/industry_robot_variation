# 로봇 산업 활용성 평가 웹앱

GitHub Pages에 바로 올릴 수 있는 정적 웹앱입니다.

## 포함 파일

- `index.html`
- `styles.css`
- `app.js`

## GitHub Pages 등록 방법

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더 안의 파일들을 저장소 루트에 업로드합니다.
3. 저장소의 `Settings` → `Pages`로 이동합니다.
4. `Deploy from a branch`를 선택합니다.
5. Branch는 `main`, 폴더는 `/root`를 선택하고 저장합니다.
6. 잠시 후 표시되는 GitHub Pages 주소로 접속합니다.
